UDP EVENTS

Auf der Gruppe FF02::6004 schickt die IceBox Multicast-Nachrichten. Die Nachrichten beinhalten einen kleinen JSON-String

Die JSON-Elemente haben die Folgende Struktur

Wenn ein Getränk gekauft wurde:

{ "eventtime":1457277798169, "eventcontent": { "eventtype":"consumption", "drink":"4260031874056" } }

Wenn sich ein neuer User registriert:

{ "eventtime":1457277798169, "eventcontent": { "eventtype":"newuser", "username":"tvluke" } }

Wenn die Daten zu einem Getränk geändert wurde:

{ "eventtime":1457277798169, "eventcontent": { "eventtype":"drinkupdate", "drink":"4260031874056" } }
